원격 저장소 - 로컬 저장소

원격 : 깃헙 사이트
로컬 :  내 컴퓨터




폴더 이동 : cd (change directory)

ls (list) : 내가 현재 위치해있는 폴더에 모든 파일과 폴더를 보여줌


git clone 원격저장소주소
-> 해당 폴더에 원격저장소를 복제(원격이랑 로컬 연결)

git checkout -b 브런치이름 -> 현재 위치해있는 브런치를 나가면서 새로운 브런치를 만들고 거기로 이동



택배

commit -> 내가 물품을 정했으면, 그 물품을 포장하는거

push  -> 발송
